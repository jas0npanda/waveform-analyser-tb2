Mini coursework - UWE - Jason Chen

How to compile and run:

1. Clone or copy repository given
2. Open repo and click the build icon in CLion
3. Copy power_quality_log.csv to cmake-build-debug folder or copy path to power_quality_log.csv
4. Open the terminal, navigating to the path: cmake-build-debug
5. Run the command: .\waveform_analyser_tb2.exe power_quality_log.csv (or path to csv)
